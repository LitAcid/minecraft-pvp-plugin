# PVP Core | All-in-One PvP Plugin (Duels, FFA, Queue)

[![PVP Core](https://raw.githubusercontent.com/LitAcid/minecraft-pvp-plugin/main/thumbnail-pvpcore.png)](https://pariah3.gumroad.com/l/zuadnl)

PVP Core is a complete **all-in-one PvP plugin for Minecraft servers** featuring duels, FFA, queue system, kit editor, and arena management in a single optimized solution.

Instead of using multiple plugins for PvP systems, PVP Core gives you everything in one lightweight and stable plugin.

---

## ⚔ Features

- Duels system for 1v1 PvP
- FFA gamemodes
- Queue system with GUI
- Kit editor for players
- Arena creation + regeneration
- Player stats system
- Core commands like /spawn and /leave
- Built-in `/guide` setup system

---

## 🧭 Easy Setup

Use:
/guide

Follow the in-game steps to quickly set up your PvP server.

---

## 🧾 Commands

**Player**
- /queue
- /ffa <gamemode>
- /challenge
- /kiteditor

**Admin**
- /kitcreate
- /kitassign <ffa|duels>
- /arena wand
- /arena create | delete | set1 | set2 | setspawn | list
